## Original Request
实现与阶级绑定的技能系统，正负值各 5 个技能（被动/主动），掉阶自动移除，丰富游戏体验。

## 已确认设计（继承自 TASK001）

### 设计原则
1. 技能随称号绑定，掉阶即失去（D7）
2. 每个阶级给一个技能（被动或主动），逐级增强
3. 正值技能偏"强化自身增长"，负值技能偏"影响他人 + 跨世界互动"
4. 主动技能无额外 CD 限制（受指令本身 CD 约束），除非效果极强
5. 允许仅对自己的被动 buff，难度递增平衡（D2）

### 正值世界技能

| 阶级 | 技能名   | 类型 | 效果                                | 平衡说明                               |
| ---- | -------- | ---- | ----------------------------------- | -------------------------------------- |
| 一阶 | 牛力加持 | 被动 | PK 获取长度 +15%                    | 基础 buff                              |
| 二阶 | 连击挑战 | 主动 | PK 可同时 @两人对决                 | 对两人独立判胜负，总获取量有上限       |
| 三阶 | 铁壁之心 | 被动 | PK 损失减少 15%                     | TASK002 方案下仍可用                   |
| 四阶 | 征服光环 | 被动 | 透群友时额外增加自己长度 +0.2~0.5cm | 需 TASK007 联动                        |
| 五阶 | 群体碾压 | 主动 | 透群友可同时指定多个目标(2~3人)     | 终极版透，社死加倍                     |

### 负值世界技能

| 阶级 | 技能名   | 类型 | 效果                                  | 平衡说明                                             |
| ---- | -------- | ---- | ------------------------------------- | ---------------------------------------------------- |
| 一阶 | 深渊共鸣 | 被动 | PK 深度获取 +15%                      | 与正值一阶对称                                       |
| 二阶 | 吸取回正 | 主动 | @正值用户，缩减对方长度，自己变为正值 | **一次性**——使用后变正值，自动失去全部负值称号和技能 |
| 三阶 | 深渊之壁 | 被动 | PK 损失减少 15%                       | 与正值三阶对称                                       |
| 四阶 | 侵蚀之力 | 被动 | 榨群友时额外减少目标长度 -0.3~0.8cm   | 需 TASK007 联动                                      |
| 五阶 | 深渊召唤 | 主动 | 榨群友可同时指定多个目标(2~3人)       | 终极版榨                                             |

### "吸取回正"详细设计

```
触发: 负值二阶完成后获取技能
使用: @一个正值用户
效果:
  1. 目标正值用户：长度 -= 自己深度绝对值 × 吸取系数（如 0.3）
     → 例：自己 -300cm，目标 50cm → 目标损失 300×0.3=90cm → 目标变为 -40cm
  2. 自己：从负值直接变为 +10cm（固定起始值，重新开始）
  3. 失去全部负值称号 + 全部负值技能
  4. 目标如果被吸到 ≤ 0，走正常的 xnn / 雌堕流程
注意: 无 CD，因为使用即"重生"，技能自动消失（D17）
```

### 设计决策
- D1: 技能获取方式 = 阶级挑战完成后获取
- D2: 允许仅对自己的被动 buff
- D7: 掉阶后取消技能
- D11: 负值主动技能"吸取回正"
- D17: 吸取回正无 CD，一次性技能
