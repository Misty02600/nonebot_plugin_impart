# [TASK003] - 美化用户数据图输出

**Status:** Completed
**Added:** 2026-03-03
**Updated:** 2026-03-03

## Original Request
美化用户数据图输出。将现有 Pillow 手绘的柱状图（排行榜）和折线图（注入历史）替换为 HTML/CSS + Jinja2 模板渲染方案，通过 `nonebot-plugin-htmlrender`（Playwright）生成精致的图片。

## 现状分析

### 当前问题
| 问题        | 描述                                           |
| ----------- | ---------------------------------------------- |
| 视觉质量低  | 白底、黑线、硬编码坐标，类似 Excel 2003 风格   |
| 硬编码布局  | 1920×1080 画布上用像素坐标定位，极难维护       |
| 图例截断    | 名字超 8 字符直接砍断 `keys[i][:8]`            |
| 颜色随机    | 每次 random.shuffle 颜色不一致，且色板仅 10 色 |
| 无自适应    | 数据条数变化时可能溢出画布                     |
| 无标题/背景 | 缺少图表标题、轴标签、美观背景                 |

### 影响范围
- `infra/draw_img.py` — DrawBarChart 类（`draw_bar_chart` + `draw_line_chart`）
- `bot/handlers/rank.py` — 调用 `chart.draw_bar_chart(data)` 生成排行榜图
- `bot/handlers/injection.py` — 调用 `chart.draw_line_chart(inject_data)` 生成注入历史图
- `bot/dependencies.py` — DrawBarChart 实例化与依赖注入

### 现有调用方式
```python
# rank.py 中:
data = {top5names[i]: top5[i]["jj_length"] for i in range(len(top5))}
for i in range(len(last5)):
    data[last5names[i]] = last5[i]["jj_length"]
img_bytes = await chart.draw_bar_chart(data)
reply = f"你的排名为{index[0] + 1}喵"
await matcher.finish(MessageSegment.image(img_bytes) + reply, at_sender=True)

# injection.py 中:
inject_data: dict[str, float] = {}
for item in data:
    inject_data[item["date"]] = item["volume"]
await chart.draw_line_chart(inject_data)
```

### 依赖注入
```python
# dependencies.py
from ..infra.draw_img import DrawBarChart
_draw_chart = DrawBarChart()
DrawChartDep = Annotated[DrawBarChart, Depends(get_draw_chart)]

# bot/__init__.py 中导出 DrawChartDep
```

## Thought Process

### 方案选择
采用 **HTML/CSS + Jinja2 + nonebot_plugin_htmlrender** 方案：
- 前端级 UI 控制力，项目已有 htmlrender 使用经验（migut-help 插件）
- migut-help 的 `render_help.py` 提供了完整的参考实现（Jinja2 + get_new_page + page.goto(file://)）

### 设计方向（最终确定）
经过 7 轮迭代，最终确定**可爱蜜桃粉风格**：

| 要素       | 决策                                                     |
| ---------- | -------------------------------------------------------- |
| 字体       | Quicksand (圆润英文) + Noto Sans SC (中文兜底)           |
| 主色       | 蜜桃粉 `#FFA7B6` + 奶油底 `#FFF3EE`                      |
| 前三名     | 金 `#C49030` / 银 `#A0A0A0` / 铜 `#B88456` 序号+头像边框 |
| 倒数区     | 天蓝 `#A8C8E8`                                           |
| 负值       | 柔红 `#D07878` / `#E8A0A0`                               |
| 查询者高亮 | 粉色加强背景 + 粉色左边框 + 粉色文字 + 加粗头像边框      |
| 卡片背景   | 半透明白 `rgba(255,255,255,0.8)` 圆角16px                |
| 列表行     | 紧凑布局，6px padding，排名+头像+昵称+小柱状条+数值      |
| Header     | 粉色底 + 白色标题，**无斜纹装饰**                        |
| 底栏       | **无 "Generated By Migut"**，仅用 padding 留白           |

### 排行榜数据布局（三段式）
```
┌─────────────────────────────┐
│  Impart 排行       共 N 人  │  ← Header
├─────────────────────────────┤
│  ▎前 5 名                   │
│  1  [头像] 大佬A   ████ 32cm│
│  2  [头像] 超人    ███  27cm│
│  3  [头像] 咕咕    ██   22cm│
│  4  [头像] 群友    ██   18cm│
│  5  [头像] 路人    █    13cm│
│  ─────────────────────────  │
│  ▎你的位置                  │  ← 仅当用户不在前5/后5时显示
│  ···                        │
│  7  [头像] 老王         10cm│
│  8  [头像] ★我自己★     8cm│  ← 高亮行（.self 样式）
│  9  [头像] 摸鱼          7cm│
│  ···                        │
│  ─────────────────────────  │
│  ▎倒数 5 名                 │
│  16 [头像] 菜鸟         3cm │
│  ...                        │
│  20 [头像] 惨惨       -5cm  │
└─────────────────────────────┘
```

**条件逻辑**：
- 如果用户在前 5 名内：高亮前 5 中对应行，不显示"你的位置"段
- 如果用户在倒数 5 名内：高亮倒数 5 中对应行，不显示"你的位置"段
- 如果用户在中间：显示"你的位置"段，包含用户和上下各 1 名邻居

### 迭代中排除的方向
- ❌ 深色主题（太暗/太丑）
- ❌ 橙色 migut-help 风（太 AI 味）
- ❌ 千篇一律的大圆角卡片
- ❌ 硬加 emoji 装饰
- ❌ 灰绿/薰衣草紫色系（太淡、不够可爱）
- ❌ 领奖台布局（太占空间）
- ❌ 纯黑/深灰文字
- ❌ Header 斜纹装饰
- ❌ "Generated By Migut" 签名

### 架构决策
1. **接口需扩展**：新渲染器不再接受简单的 `dict[str, float]`，需要传入结构化数据（头像URL、昵称、数值、排名、是否高亮）
2. **模板目录结构**：在 `infra/templates/charts/` 下放 Jinja2 模板和 CSS
3. **使用 `get_new_page`**：与 migut-help 一致，用 `page.goto(file://)` 方式加载
4. **类名重命名**：`DrawBarChart` → `ChartRenderer`，同步更新依赖注入
5. **字体加载**：Quicksand + Noto Sans SC 通过 CSS `@font-face` 加载本地 OTF/TTF 文件（参考 migut-help 的字体加载方式），或通过 Google Fonts CDN

### 2. 实现 chart_renderer.py
参考 `migut-help/render_help.py` 的实现模式：
```python
class ChartRenderer:
    async def draw_bar_chart(self, rank_data: list[RankEntry], user_rank: int, total: int) -> bytes:
        """渲染排行榜图片

        Args:
            rank_data: 包含 top5 + 用户周围 + bottom5 的排名数据
            user_rank: 查询者的排名（1-indexed）
            total: 总人数
        """
        ...

    async def draw_line_chart(self, data: dict[str, float], user_name: str, user_avatar: str, total: float) -> bytes:
        """渲染注入历史折线图

        Args:
            data: {日期: 注入量}
            user_name: 用户昵称
            user_avatar: 用户头像URL
            total: 累计总量
        """
        ...
```

关键步骤（参考 migut-help）：
1. Jinja2 渲染 HTML 字符串
2. 写入临时 HTML 文件到模板目录下（使 CSS/字体的相对路径生效）
3. `get_new_page(device_scale_factor=2)` 打开页面
4. `page.goto(file_url, wait_until="networkidle")`
5. 根据内容高度自适应 viewport
6. `page.screenshot(full_page=True, type="png")` 截图
