## Design: 技能系统（第二版）

### 设计原则
1. 技能随称号绑定，掉阶即失去
2. 每个阶级给一个技能（被动或主动），逐级增强
3. 正值技能偏"强化自身增长"，负值技能偏"影响他人 + 跨世界互动"
4. 主动技能无额外 CD 限制（受指令本身 CD 约束），除非效果极强

### 正值世界技能

| 阶级 | 技能名   | 类型 | 效果                                | 平衡说明                             |
| ---- | -------- | ---- | ----------------------------------- | ------------------------------------ |
| 一阶 | 牛力加持 | 被动 | PK 获取长度 +15%                    | 基础 buff，挑战难度已限制            |
| 二阶 | 连击挑战 | 主动 | PK 可同时 @两人对决                 | 对两人都独立判胜负，但总获取量有上限 |
| 三阶 | 铁壁之心 | 被动 | PK 损失减少 15%                        | ✅ (TASK002-R3) 方案 D 下仍可用，或改为"胜率变化 ±1%→±0.5%"，待确认 |
| 四阶 | 征服光环 | 被动 | 透群友时额外增加自己长度 +0.2~0.5cm | 新增增长途径                         |
| 五阶 | 群体碾压 | 主动 | 透群友可同时指定多个目标(2~3人)     | 终极版透，社死加倍                   |

### 负值世界技能

| 阶级 | 技能名   | 类型 | 效果                                  | 平衡说明                                             |
| ---- | -------- | ---- | ------------------------------------- | ---------------------------------------------------- |
| 一阶 | 深渊共鸣 | 被动 | PK 深度获取 +15%                      | 与正值一阶对称                                       |
| 二阶 | 吸取回正 | 主动 | @正值用户，缩减对方长度，自己变为正值 | **一次性**——使用后变正值，自动失去全部负值称号和技能 |
| 三阶 | 深渊之壁 | 被动 | PK 损失减少 15%                        | ✅ (TASK002-R3) 方案 D 下仍可用，与正值三阶对称 |
| 四阶 | 侵蚀之力 | 被动 | 榨群友时额外减少目标长度 -0.3~0.8cm   | 强化跨世界干扰能力                                   |
| 五阶 | 深渊召唤 | 主动 | 榨群友可同时指定多个目标(2~3人)       | 终极版榨，恐怖的深渊之力                             |

### "吸取回正"详细设计

```
触发: 负值二阶完成后获取技能
使用: @一个正值用户
效果:
  1. 目标正值用户：长度 -= 自己深度的绝对值 × 吸取系数（如 0.3）
     → 例：自己 -300cm，目标 50cm → 目标损失 300×0.3=90cm → 目标变为 -40cm
  2. 自己：从负值直接变为 +10cm（固定起始值，重新开始）
  3. 失去全部负值称号 + 全部负值技能
  4. 目标如果被吸到 ≤ 0，走正常的 xnn / 雌堕流程
注意: 无 CD，因为使用即"重生"，技能自动消失
```

## Design: 扩展 Buff / 技能系统（阶级之外）

除阶级称号技能外，引入多种 buff 获取方式来丰富日常体验。

### 一、每日运势（每日 Buff）

**机制**：每天第一次执行任何指令时，自动触发"今日运势"，随机获得一个持续到当日结束的 buff/debuff。

| 运势   | 概率 | 效果                          | 风味文案                            |
| ------ | ---- | ----------------------------- | ----------------------------------- |
| 🌟 大吉 | 5%   | 今日所有增长/挖深 ×1.5        | "今日天降祥瑞，你的牛牛蠢蠢欲动..." |
| ✨ 中吉 | 15%  | PK 获取 +25%                  | "微风拂面，你感到一阵自信..."       |
| 🍀 小吉 | 20%  | 所有 CD 减半                  | "精力充沛的一天！"                  |
| ☁️ 平   | 30%  | 无特殊效果                    | "平平无奇的一天。"                  |
| 🌧️ 小凶 | 15%  | 所有增长/挖深 ×0.75           | "今日状态不佳..."                   |
| ⛈️ 凶   | 10%  | PK 败方损失额外 +25%          | "感觉要遭..."                       |
| 💀 大凶 | 5%   | 打胶/开扣 有 30% 概率反向变化 | "天降横祸，不如早点回家躺平！"      |

**平衡说明**：
- 正面效果(大吉+中吉+小吉) 40% vs 负面效果(小凶+凶+大凶) 30%，略偏正面
- 大凶的"反向变化"很有戏剧性——你打胶结果变短了！
- 运势可以通过指令 `今日运势` / `运势查询` 主动查看

### 三、连续/连击系统（Streak）

**机制**：连续行为累积奖励，激励持续参与。

#### PK 连胜/连败

```
连胜:
  3连胜 → 下次 PK 获取 +25%      "气势如虹！"
  5连胜 → 下次 PK 获取 +50%      "五连胜！势不可挡！"
  7连胜 → 下次 PK 获取 +100%     "七连胜！！神挡杀神！"
  (连胜被打断后重置)

连败:
  3连败 → 下次 PK 胜率临时 +10%   "屡败屡战..."
  5连败 → 下次 PK 胜率临时 +20%   "背水一战！"
  7连败 → 下次 PK 必定胜利        "触底反弹！命运的扭转！"
  (连败被打断后重置)
```

**平衡说明**：
- 连胜奖励是**增量 buff**（赢了获得更多）而非胜率提升，不改变胜负结果
- 连败保护是**胜率临时提升**，帮助翻盘，防止螺旋下降
- 7 连败必胜是最终兜底，防止极端负体验
- 连胜/连败只在连续 PK 中计数（穿插其他操作不影响）

#### 每日签到连续

```
连续签到（每天至少操作一次）:
  3天连续 → 当天打胶/开扣额外 +0.5cm
  7天连续 → 当天首次 PK 获取 ×2
  14天连续 → 解锁当天特殊指令"命运对决"（PK 双方赌注翻倍）
  30天连续 → 永久称号前缀"常驻の"（不影响数值，纯装饰）
  断签后从头开始
```

### 四、条件触发被动（Conditional Passives）

**机制**：满足特定条件时自动启用，不需要手动触发。与阶级技能不同，这些是**所有玩家通用**的。

| 效果名       | 触发条件                            | 效果                   | 设计理由                 |
| ------------ | ----------------------------------- | ---------------------- | ------------------------ |
| 🔥 濒死反击   | 长度 0~5cm (xnn)                    | PK 获胜时增量 ×2       | 给 xnn 玩家翻身希望      |
| 🗡️ 以下犯上   | PK 对手长度/深度 > 自己 ×2          | 胜率临时 +8%           | 弱者有挑战强者的动力     |
| 🛡️ 居高临下   | PK 对手长度/深度 < 自己 ×0.5        | 失败时损失减半         | 强者不会被弱者"偷鸡"崩盘 |
| 💀 深渊凝视   | 深度 > 100cm                        | 榨群友侵蚀效果 +0.1cm  | 越深的人越有威慑力       |
| 🌟 初心者守护 | 首次注册后 24 小时内                | 所有损失减半           | 新手保护期               |
| 🔁 复仇之刃   | 上一次 PK 输给了某人                | 对同一人 PK 时胜率 +5% | 激励复仇，增加恩怨感     |
| 💫 背水一战   | 处于挑战中且距离目标 < 总差距的 10% | PK 获取 +30%           | 快冲线时的最后冲刺 buff  |

**平衡说明**：
- 每天最多触发 1 个事件（也可能不触发，如 30% 概率无事件）
- 正面/负面事件比例约 4:3
- "双倍日"最刺激但也最危险——增减都翻倍

## Design: 透群友 × 长度/概率联动

### 核心问题
透群友目前只产生"注入记录"，与核心数值（长度/概率）完全脱离。需要联动。

### 正值世界：透群友

| 条件       | 对自己              | 对目标       |
| ---------- | ------------------- | ------------ |
| 透任意群友 | **+0.1~0.3cm 长度** | 保持注入效果 |

正值世界=增长自己，透是"征服"。

### 负值世界：榨群友（深度比较机制）✨

**核心规则：效果取决于自己与目标的深度对比**

```
你（负值玩家）使用 榨群友 → 目标是谁？

┌─ 目标是正值玩家（或比自己浅的负值玩家）
│   → 你比对方"深" → 侵蚀：减少目标长度/深度 -0.1~0.5cm
│   → "以深渊之力侵蚀了TA的牛牛..."
│
└─ 目标是比自己更深的负值玩家
    → 对方比你"深" → 汲取：增加自己深度 +0.1~0.5cm
    → "你从TA的深渊中汲取了力量，变得更深了..."
```

**判定规则（绝对值比较）：**

| 你的状态 | 目标状态 | 深度对比 | 效果                     |
| -------- | -------- | -------- | ------------------------ |
| -50cm    | +30cm    | 你更深   | 减目标长度               |
| -50cm    | -20cm    | 你更深   | 减目标深度（让对方变浅） |
| -10cm    | -50cm    | 对方更深 | 加自己深度（自己变深）   |
| -5cm     | -100cm   | 对方更深 | 加自己深度（快速追赶）   |

**设计精妙之处：**
1. **新人追赶**：刚雌堕的 -5cm 用户，面前几乎所有负值玩家都"比自己深"→ 榨他们就能快速增加自己深度，不会在新手期卡住
2. **深渊生态**：深度越深的玩家，能被浅玩家"汲取"但无法从他们身上获得深度 → 深度大佬需要去侵蚀正值玩家来发挥优势
3. **跨世界张力**：深度大佬的主要"猎物"是正值玩家，浅负值玩家的"营养源"是深负值玩家 → 形成食物链

### 与技能的叠加

有了四阶/五阶技能后：
- 正值四阶"征服光环"：透群友自己获取从 +0.1~0.3 提升到 +0.2~0.5
- 负值四阶"侵蚀之力"：榨群友的侵蚀/汲取数值从 ±0.1~0.5 提升到 ±0.3~0.8
- 正值五阶"群体碾压"：可以透多人，每人都给自己加长度
- 负值五阶"深渊召唤"：可以榨多人，每人都按深度比较规则分别计算效果
