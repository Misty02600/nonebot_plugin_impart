## Original Request
实现透群友/榨群友与核心数值（长度/深度）的联动，包括正值透→自己+长度、负值榨→深度比较机制（食物链），以及技能增强叠加。

## 已确认设计（继承自 TASK001）

### 正值世界：透群友

| 条件       | 对自己              | 对目标       |
| ---------- | ------------------- | ------------ |
| 透任意群友 | **+0.1~0.3cm 长度** | 保持注入效果 |

正值世界 = 增长自己，透是"征服"。

### 负值世界：榨群友（深度比较机制）

**核心规则：效果取决于自己与目标的深度对比（D19）**

```
你（负值玩家）使用 榨群友 → 目标是谁？

┌─ 目标是正值玩家（或比自己浅的负值玩家）
│   → 你比对方"深" → 侵蚀：减少目标长度/深度 -0.1~0.5cm
│   → "以深渊之力侵蚀了TA的牛牛..."
│
└─ 目标是比自己更深的负值玩家
    → 对方比你"深" → 汲取：增加自己深度 +0.1~0.5cm
    → "你从TA的深渊中汲取了力量，变得更深了..."
```

**判定规则（绝对值比较）：**

| 你的状态 | 目标状态 | 深度对比 | 效果                     |
| -------- | -------- | -------- | ------------------------ |
| -50cm    | +30cm    | 你更深   | 减目标长度               |
| -50cm    | -20cm    | 你更深   | 减目标深度（让对方变浅） |
| -10cm    | -50cm    | 对方更深 | 加自己深度（自己变深）   |
| -5cm     | -100cm   | 对方更深 | 加自己深度（快速追赶）   |

**食物链设计精妙之处：**
1. **新人追赶**：刚雌堕的 -5cm 用户，面前几乎所有负值玩家都"比自己深"→ 榨他们就能快速增加自己深度
2. **深渊生态**：深度越深的玩家，能被浅玩家"汲取"但无法从他们身上获得深度 → 深度大佬需要去侵蚀正值玩家
3. **跨世界张力**：深度大佬的主要"猎物"是正值玩家，浅负值玩家的"营养源"是深负值玩家 → 形成食物链

### 与技能的叠加（需 TASK006 完成后）

- 正值四阶"征服光环"：透群友自己获取从 +0.1~0.3 提升到 +0.2~0.5
- 负值四阶"侵蚀之力"：榨群友效果从 ±0.1~0.5 提升到 ±0.3~0.8
- 正值五阶"群体碾压"：可以透多人，每人都给自己加长度
- 负值五阶"深渊召唤"：可以榨多人，每人都按深度比较规则分别计算效果

### 设计决策
- D12: 榨群友效果 = 长期缓慢减少目标正值玩家的长度
- D18: 透群友需要与长度/概率机制联动
- D19: 榨群友深度比较机制（榨比自己浅→侵蚀，榨比自己深→汲取）
- D20: 待确认——透群友联动方向的最终数值
