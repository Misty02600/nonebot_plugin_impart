**Status:** Pending
**Added:** 2026-03-04
**Updated:** 2026-03-04
**Origin:** 从 TASK001 拆分
**Depends:** TASK004（负值世界基础）— 弱依赖，大部分子系统可独立开发

## Original Request
实现阶级技能之外的多种 Buff 获取方式，包括每日运势、随机触发 Proc、连击系统、条件触发被动、全群事件，丰富日常游戏体验。

## 已确认设计（继承自 TASK001）

### 系统总览

```
扩展 Buff 获取方式:

├── 每日运势（随机日 buff，当日有效）
│   └── 大吉 / 中吉 / 小吉 / 平 / 小凶 / 凶 / 大凶
│
├── 随机触发 Proc（操作时概率触发）
│   ├── PK: 暴击 / 闪避 / 反转 / 双杀 / 和局
│   └── 打胶嗦: 精准一击 / 用力过猛 / 心意相通
│
├── 连击系统（连续行为累积）
│   ├── PK 连胜: +25% → +50% → +100%
│   ├── PK 连败: 胜率+10% → +20% → 必胜
│   └── 签到连续: 3天/7天/14天/30天 各有奖励
│
├── 条件触发被动（满足条件自动生效）
│   └── 濒死反击 / 以下犯上 / 居高临下 / 深渊凝视 / 初心者守护 / 复仇之刃 / 背水一战
│
└── 全群事件（随机全群效果）
    └── 生长季 / PK嘉年华 / 深渊潮汐 / 万物凋零 / 混沌之日 / 休战日 / 双倍日
```

### 一、每日运势

| 运势   | 概率 | 效果                          |
| ------ | ---- | ----------------------------- |
| 🌟 大吉 | 5%   | 今日所有增长/挖深 ×1.5        |
| ✨ 中吉 | 15%  | PK 获取 +25%                  |
| 🍀 小吉 | 20%  | 所有 CD 减半                  |
| ☁️ 平   | 30%  | 无特殊效果                    |
| 🌧️ 小凶 | 15%  | 所有增长/挖深 ×0.75           |
| ⛈️ 凶   | 10%  | PK 败方损失额外 +25%          |
| 💀 大凶 | 5%   | 打胶/开扣 有 30% 概率反向变化 |

### 二、随机触发 Proc

#### PK Proc

| 效果名 | 触发时机  | 概率 | 效果                     |
| ------ | --------- | ---- | ------------------------ |
| ⚡ 暴击 | PK 胜利   | 8%   | 本次获取长度 ×2          |
| 🛡️ 闪避 | PK 失败   | 8%   | 本次损失减为 0           |
| 🔄 反转 | PK 判定前 | 3%   | 本次胜负结果反转         |
| 💥 双杀 | PK 胜利   | 5%   | 赢方获得全额 num 而非 num/2 |
| 🤝 和局 | PK 判定前 | 3%   | 双方都不增不减           |

#### 打胶/嗦/开扣/舔 Proc

| 效果名     | 触发时机  | 概率 | 效果                   |
| ---------- | --------- | ---- | ---------------------- |
| 🎯 精准一击 | 打胶/开扣 | 10%  | 增量翻倍               |
| 😱 用力过猛 | 打胶/开扣 | 5%   | 增量变为负（反向变化） |
| 💞 心意相通 | 嗦/舔     | 8%   | 目标和自己都获得增量   |

### 三、连击系统

#### PK 连胜/连败

| 连续次数 | 连胜效果               | 连败效果               |
| -------- | ---------------------- | ---------------------- |
| 3连      | 下次 PK 获取 +25%      | 下次 PK 胜率临时 +10%  |
| 5连      | 下次 PK 获取 +50%      | 下次 PK 胜率临时 +20%  |
| 7连      | 下次 PK 获取 +100%     | 下次 PK 必定胜利       |

#### 每日签到连续

| 连续天数 | 奖励                                         |
| -------- | -------------------------------------------- |
| 3天      | 当天打胶/开扣额外 +0.5cm                     |
| 7天      | 当天首次 PK 获取 ×2                           |
| 14天     | 解锁当天特殊指令"命运对决"（赌注翻倍）       |
| 30天     | 永久称号前缀"常驻の"（纯装饰）               |

### 四、条件触发被动

| 效果名       | 触发条件                            | 效果                  |
| ------------ | ----------------------------------- | --------------------- |
| 🔥 濒死反击   | 长度 0~5cm (xnn)                    | PK 获胜增量 ×2        |
| 🗡️ 以下犯上   | 对手 > 自己 ×2                      | 胜率临时 +8%          |
| 🛡️ 居高临下   | 对手 < 自己 ×0.5                    | 失败损失减半          |
| 💀 深渊凝视   | 深度 > 100cm                        | 榨群友侵蚀 +0.1cm    |
| 🌟 初心者守护 | 首次注册后 24h 内                   | 所有损失减半          |
| 🔁 复仇之刃   | 上次 PK 输给某人                    | 对同人 PK 胜率 +5%    |
| 💫 背水一战   | 挑战中距目标 < 总差距 10%           | PK 获取 +30%          |

### 五、全群事件

| 事件名      | 效果                              | 概率   |
| ----------- | --------------------------------- | ------ |
| 🌸 生长季    | 所有增量 +30%                      | ~14%   |
| ⚔️ PK 嘉年华 | PK CD 减半，获取量 +20%            | ~14%   |
| 🌊 深渊潮汐  | 负值深度增长 +30%                  | ~14%   |
| ❄️ 万物凋零  | 所有增量 -20%                      | ~14%   |
| 🎲 混沌之日  | PK 结果 15% 概率反转               | ~8%    |
| ☮️ 休战日    | PK 不可用，打胶/嗦 CD 减半         | ~8%    |
| 💰 双倍日    | 所有数值变化 ×2（增减都翻倍）      | ~8%    |
| 无事件       |                                   | ~20%   |

## Implementation Plan

- [ ] 8.1 Buff 框架核心设计
  - 定义通用 Buff 接口（类型、持续时间、效果参数、来源）
  - Buff 管理器（添加/移除/查询/过期清理）
  - Buff 效果应用中间件（在各操作中注入）
- [ ] 8.2 每日运势系统
  - 每日首次操作触发运势抽取
  - 运势 buff 当日有效，次日清除
  - `今日运势` / `运势查询` 指令
- [ ] 8.3 Proc 随机触发系统
  - PK Proc（暴击/闪避/反转/双杀/和局）
  - 打胶/嗦 Proc（精准/用力过猛/心意相通）
  - 触发时的特殊播报
- [ ] 8.4 连击系统
  - PK 连胜/连败计数与奖励
  - 连续签到计数
  - 数据持久化（连击状态需存 DB）
- [ ] 8.5 条件触发被动
  - 各条件检测逻辑
  - 自动启用/停用
  - 复仇之刃需要记录"上次输给谁"
- [ ] 8.6 全群事件系统
  - 每日定时任务：随机选择事件或无事件
  - 群内播报事件生效
  - 事件效果全天有效
- [ ] 8.7 Buff 叠加与优先级
  - 多个 buff 同时生效时的计算顺序
  - 防止极端叠加（如大吉 + 连胜 + 暴击 = ×2 × ×1.5 × ×2 = ×6）
  - 考虑是否需要叠加上限
